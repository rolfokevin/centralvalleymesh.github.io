// What is this thing?

A Meshtastic LoRa radio node.

This device is a small radio transmitter that's part of the Central Valley Mesh — a volunteer-run communication network built on Meshtastic open-source software.

It uses long-range radio (LoRa) to pass encrypted text messages, GPS positions, and sensor data between nodes — with no internet, no cell towers, and no monthly fees. Messages hop from device to device across the valley.

Think of it like a neighborhood walkie-talkie network, but it can reach 20–30+ miles and is encrypted by default.

Please don't remove or tamper with this device.

This node is privately owned and placed with permission. It's part of a community communication network. If there's a problem with its placement, please reach out — contact info is below.
